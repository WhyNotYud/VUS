# 🏫 VUS Website Project

![VUS Logo](./image/logo.png)

> Static website for VUS English Centers (Anh Văn Hội Việt Mỹ)

## 🚀 Features

- **Responsive header** with logo and navigation
- **Program sections** for different age groups:
  - 👶 SmartKids (4-6 tuổi)
  - 🧒 SuperKids (6-11 tuổi)
  - 🧑 Young Leaders (11-15 tuổi)
  - 🎓 EIC (17+ tuổi)
- 📰 News & events section
- 📞 Contact information
